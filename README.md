# Rosebud Apothecary — Storefront
 
A single-file, self-contained storefront (`index.html`) — no build step, no backend, no dependencies to install. Everything (catalog, filtering, cart, and a simulated checkout) runs client-side in vanilla JS.
 
## Run it locally
Just double-click `index.html`, or serve it:
```bash
python3 -m http.server 8000
```
then open `http://localhost:8000`.
